# Tic-Tac-Toe Python Game - Introduction to Computing 
A beginner Python project from my freshman-year Introduction to Computing course, built to practice core programming fundamentals.
## How to Run
 
1. Clone the repository:
   ```
   git clone https://github.com/JaneDalessio/tic-tac-toe-python.git
   ```
 
2. Make sure you have `graphics.py` in the same directory. It can be downloaded from [John Zelle's website](https://mcsp.wartburg.edu/zelle/python/).
3. Run the game:
   ```
   python "Tic-Tac-Toe Project.py"
   ```
 
> **Note:** `graphics.py` requires a display environment (it will not run headlessly). It works out of the box on Windows and macOS. On Linux, ensure a display is available.
